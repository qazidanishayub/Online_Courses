Assignment Overview
This assignment prefaces later work with graph traversal problems. As preparation, you will cover ways to represent basic graphs and trees using base Python and the networkx library. The main goals in this portion of the assignment are:

To represent and manipulate graphs & trees using Python idioms
To visualize graphs and trees using Python
To perform uniformed search in graphs using Python
